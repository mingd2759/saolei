1. Download the repository.
2. Open the `index.html` file in your web browser.

## Game Rules
1. The objective is to clear a minefield without detonating any mines.
2. Click on squares to uncover them; if you uncover a mine, the game is over.
3. If you uncover a number, it represents how many adjacent squares contain mines.

## Operation Instructions
- Use left-click to uncover a square.
- Right-click to flag a suspected mine.

## Difficulty Settings
- Beginner: 8x8 grid with 10 mines.
- Intermediate: 16x16 grid with 40 mines.
- Expert: 24x24 grid with 99 mines.
